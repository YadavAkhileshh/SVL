# SVL - Smart Video Learner

AI-Powered Educational Platform for Interactive Learning

Transform any YouTube educational video into comprehensive study materials with AI-generated flashcards, quizzes, and personalized tutoring.

## Overview

SVL is an intelligent learning platform that converts YouTube educational videos into interactive study materials. Using advanced AI models, it generates comprehensive summaries, flashcards, quizzes, and provides personalized tutoring to enhance the learning experience.

## Core Features

### Video Processing
Process any YouTube educational video to extract transcripts and generate AI-powered study materials. The system automatically detects the topic and creates comprehensive learning content tailored to the subject matter.

### Study Materials Generation
- Video summaries providing concise overviews of the content
- Detailed explanations covering all key concepts in depth
- Key learning points highlighting the most important takeaways
- Full transcript access for reference

### Interactive Flashcards
Study with intelligent flashcards in multiple modes:
- List Mode for quick review and overview
- Deck Mode with 3D flip animations and mastery tracking
- Game Mode featuring four interactive learning games
- Generate additional flashcards on demand

Flashcard games include drag and drop matching, memory match with multiple difficulty levels, speed quiz with streak tracking, and type answer validation.

### Adaptive Quizzes
Take customizable quizzes with adjustable difficulty levels and question counts. The system generates multiple choice and true/false questions based on the video content. Review wrong answers and generate additional questions to reinforce learning.

### AI Tutor
Get instant help from an AI tutor that understands the context of your study material. Ask questions and receive clear explanations with real-world examples, step-by-step breakdowns, and visual formula rendering.

### Learning Roadmap
Generate personalized learning paths for any topic. The system creates structured roadmaps with curated YouTube videos, tracks your progress, and allows you to mark topics as completed. Chat with AI about specific topics and get video summaries.

### Code Learning Roadmap
Follow structured programming learning paths for Python, JavaScript, Java, and custom languages. Each roadmap includes topic-based progression, video tutorials, practice sites, learning resources, and progress tracking.

### Progress Tracking
Monitor your learning journey with comprehensive statistics including videos processed, quizzes taken, flashcards studied, study sessions completed, and current learning streaks. Unlock achievements as you progress.

### User Authentication
Secure authentication with email/password signup and Google Sign-In integration. User profiles store all learning data and progress.

### Dark Mode
Full dark mode support with smooth transitions and consistent theming across all pages. The interface uses yellow and orange colors in light mode, purple and indigo in dark mode.

### History Management
Access all your past study sessions, revisit previous materials, regenerate content, and manage your learning history.

## Technology

Built with React for the frontend and FastAPI for the backend. Uses Firebase for authentication and data storage. Powered by Groq API with Llama 3.3 70B for AI content generation.

## Use Cases

- Students studying from YouTube educational videos
- Self-learners wanting structured study materials
- Educators creating supplementary learning content
- Anyone looking to enhance video-based learning with interactive tools

## Purpose

Final year computer science project demonstrating AI integration in educational technology. Created to help students worldwide learn more effectively from online video content.
